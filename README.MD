# ДежаВю: Язык
Русский | [English](README_EN.MD)

Этот репозиторий содержит исходный код компилятора языка проекта ДежаВю. На данный момент здесь также содержится временная имплементация виртуальной машины для языка, которая в будущем будет переписана с нуля и вынесена в отдельный репозиторий. Для полноценной работы с языком в будущем потребуется установка и настройка ДежаВю: Платформа.
