# DejaVu: Language
[Русский](README.MD) | English

This repository contains the source code of language compiler of DejaVu Project. At this moment, the temporary virtual machine implementation is also contained in this repository, which will be reimplemented from scratch in future and moved to a separate repository. To work with the language properly, an installation of DejaVu: Platform will be required in the future.
